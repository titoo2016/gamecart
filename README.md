# Codenames Game - Vercel Deployment

لعبة Codenames متعددة اللاعبين مع دعم الوقت الفعلي باستخدام Socket.IO.

## المتطلبات

- Node.js 20.x أو أحدث
- npm أو pnpm

## البنية

المشروع يتكون من جزأين:

### 1. الواجهة الأمامية (Frontend)
- **المسار**: `./codenames-game/`
- **التقنيات**: React + TypeScript + Vite + TailwindCSS
- **الميزات**: واجهة مستخدم حديثة مع دعم الأجهزة المحمولة

### 2. الخادم (Backend)
- **المسار**: `./api-server/`
- **التقنيات**: Express.js + Socket.IO + TypeScript
- **الميزات**: إدارة الغرف والألعاب في الوقت الفعلي

## التثبيت والتطوير المحلي

### 1. تثبيت المتطلبات

```bash
# تثبيت متطلبات الخادم
cd api-server
npm install
cd ..

# تثبيت متطلبات الواجهة الأمامية
cd codenames-game
npm install
cd ..
```

### 2. تشغيل الخادم

```bash
cd api-server
npm run dev
# الخادم سيعمل على http://localhost:3000
```

### 3. تشغيل الواجهة الأمامية (في terminal منفصل)

```bash
cd codenames-game
PORT=5173 BASE_PATH=/ npm run dev
# الواجهة الأمامية ستعمل على http://localhost:5173
```

## النشر على Vercel

### الطريقة الأولى: استخدام Vercel CLI

```bash
# تثبيت Vercel CLI
npm install -g vercel

# النشر
vercel
```

### الطريقة الثانية: ربط مع GitHub

1. ادفع المشروع إلى GitHub
2. اذهب إلى [vercel.com](https://vercel.com)
3. انقر على "New Project"
4. اختر المستودع الخاص بك
5. اترك الإعدادات الافتراضية (سيتم استخدام `vercel.json`)
6. انقر على "Deploy"

## كيفية اللعب

1. **إنشاء غرفة جديدة**: اضغط على "Create Room" وأدخل اسمك
2. **الانضمام إلى غرفة**: أدخل رمز الغرفة واسمك
3. **اختيار الفريق**: اختر الفريق الأحمر أو الأزرق
4. **اختيار الدور**: اختر ما إذا كنت تريد أن تكون "Spymaster" أم لاعباً عادياً
5. **بدء اللعبة**: عندما يكون لديك لاعبين كافيين، اضغط "Start Game"
6. **اللعب**: اتبع القواعد القياسية للعبة Codenames

## المشاكل الشائعة والحلول

### المشكلة: الاتصال بـ Socket.IO لا يعمل

**الحل**: تأكد من أن الخادم يعمل على المنفذ الصحيح وأن الواجهة الأمامية تتصل بالعنوان الصحيح.

### المشكلة: الواجهة الأمامية لا تبني بشكل صحيح

**الحل**: تأكد من أن جميع المتغيرات البيئية مضبوطة بشكل صحيح:
```bash
PORT=5173 BASE_PATH=/ npm run build
```

### المشكلة: الخادم يتعطل بعد فترة

**الحل**: تأكد من أن جميع المتطلبات مثبتة بشكل صحيح وأن لديك إصدار Node.js المدعوم.
